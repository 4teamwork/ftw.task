    >>> self.portal.manage_setLocalRoles('hugoboss', ('Contributor', 'Editor'))

then we create a task, and set 'hugoboss' as responsible
    >>> self.portal.invokeFactory('Task', 't2')

    >>> t2.setResponsibility(['hugoboss'])
